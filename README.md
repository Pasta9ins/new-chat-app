- One-on-One chat

- Global state management with Zustand
- 20+ Theme modes
- Fully responsive UI


Tech Stack:

**Frontend:** React, TailwindCSS, DaisyUI, Axios  
**Backend:** Node.js, Express.js, MongoDB  
**Real-time:** Socket.IO  
**Authentication:** JWT  
**Deployment:** Render
